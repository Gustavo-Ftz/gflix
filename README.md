# gflix
Uma recriação da interface da Netflix para estudo e portfólio.

Desenvolvido em **HTML, e CSS**, com foco em treinar layout, responsividade e componentes visuais inspirados no serviço de streaming.

---

## Sobre o projeto  

Este projeto é um clone da interface da página inicial da Netflix, seguindo o visual moderno da plataforma.

---

## Funcionalidades  

- Layout inspirado na página inicial da Netflix  
- Interface responsiva (falta alguns ajustes)   

---

## Tecnologias usadas  

- **HTML5**  
- **CSS3**
